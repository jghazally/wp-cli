wp-user-add_to_blog(1) -- Add an existing WordPress user to an existing blog
====

## SYNOPSIS

`wp user add_to_blog` <user-login> [--role=<role>]

## OPTIONS

* `<user-login>`:

	The login of the user to add to the blog.

* `--role`=<role>:

	Add the user with the specified role. Defaults to blog default.

## EXAMPLES

	wp user add_to_blog bob --role=author
